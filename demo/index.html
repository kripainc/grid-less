Features

  • 100% markup free grids

    Grid LESS doesn't require any changes to your markup to get started. In fact, no class names are even generated for you to use! This is the beauty of LESS - you target elements in the style definition rather than the element definition.

  • 960.gs based

    Uses all of the familiar concepts from our favorite CSS grid system. This is what we would still be using if we hadn't come up with Grid LESS.
